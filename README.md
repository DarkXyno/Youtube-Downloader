Yui is a lightweight desktop video downloader built for speed, simplicity, and modern usability.

Download videos, playlists, or audio quickly with a clean interface and minimal friction.

---

## Features

- Download videos via URL
- MP4, MKV, and MP3 output formats
- Playlist download support
- Download queue manager
- Download history tracking
- File browser panel for downloaded content
- Customizable background image
- Clean modern UI
- Windows installer support

---

## Installation (Windows)

1. Go to **Releases**
2. Download the latest `YuiSetup.exe`
3. Run installer
4. Launch from Start Menu or Desktop

No Python installation required.

---

## Usage

1. Paste video or playlist URL.
2. Select format.
3. Click **Download**.
4. Files appear in chosen download folder.
5. History and files are accessible inside the app.
